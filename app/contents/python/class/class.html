クラス

classを用いてクラスを定義できます。

例ではPersonクラスのインスタンスを生成し、変数pに代入しています。

__init__メソッド

インスタンス生成の際の初期化処理は__init__という名前の特別なメソッド定義が必要です。

この__init__メソッドの引数はインスタンス生成時に渡すことができます。

インスタンスオブジェクト

メソッドの引数にあるselfはそのインスタンスオブジェクト自身を指します。

したがって、そのインスタンス固有の値はselfから参照できます。

クラス変数

クラスに固有の値をもたせることができます。これはインスタンス変数とは異なり、同クラスに由来するインスタンス全てに共通する値です。
