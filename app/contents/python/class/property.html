プロパティ

クラスの属性に対して、"getter" や "setter" の機能を付けることができます。

プロパティは、属性値への外部からのアクセスを制限することに有用です。

getter

属性値の取得を定義します。

@propertyデコレータはプロパティの関数名をそのまま "getter" にします。

setter

属性値の設定を定義します。

@{プロパティ名}.setterデコレータで "setter" を記述します。

関数名は必ず元のプロパティと同じ名前にする必要があります。

deleter

属性値の削除を定義します。

@{プロパティ名}.deleterデコレータで "deleter" を記述します。

"setter" 同様、関数名は必ず元のプロパティと同じ名前にします。
