for 文

C 言語などのfor文とは少し異なり、for A in Bのように記述します。

B には反復可能オブジェクト（iterable object）を渡し、A は要素を表します。

「イテラブルなオブジェクト」とは要素を一度に 1 つずつ返せるオブジェクトです。例として、文字列やリストなどのシーケンス型やイテレータなどがあります。


break と continue

while 文同様、break, continueを使用できます。


for ループのテクニック

range 関数

算術的な数列であるrangeオブジェクトを返します。

range(stop)あるいはrange(start, stop, [step])

start, stop, step の値のみを保存するため、リストやタプルを新しく作るよりメモリ効率が良いです。

例えば、for i in range(1, 10):とした場合、ループごとに i が 1 から 9 まで変化します。stop の値は含まれないことに注意しましょう。

start を省略した場合は、0 から始まります。


enumerate 関数

渡されたイテラブルなオブジェクトに対し、要素とそのインデックスのタプルを返します。

enumerate(iterable, start=0)

インデックスと要素を両方用いたい場合などに便利です。

for i, value in enumerate(['a', 'b', 'c']):のようにすれば、i がインデックス、value が要素（この場合'a', 'b', 'c'）となります。


zip 関数

二つ以上のシーケンスに対し同時にループを回したい場合に使います。


辞書型のループ

辞書型のキー、値のペアをfor ループで回したい場合は、辞書型のメソッドitems()を用います。

for key, value in d.items():のようにすれば、key がキー、value が値となります。
