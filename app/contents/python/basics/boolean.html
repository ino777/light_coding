真偽値

Python ではオブジェクトをデフォルトで真（True）と判定します。

偽（False）と判定されるオブジェクトは以下の通りです。

  • None, False
  • 0, 0.0, 0j, Decimal(0), Fraction(0, 1)
  • '', (), [], {}, set(), range(0)

比較

およそ他の言語と同様に==（等しい）, !=（等しくない）, <（より小さい）, <=（以下）, >（より大きい）, >=（以上）がある他、is（同一オブジェクトである）, is not（同一オブジェクトでない）も存在します。


ブール演算

Python におけるブール演算として優先順位順にnot, and, orがあります。

それぞれ集合論における「否定」,「かつ」,「または」に相当します。
